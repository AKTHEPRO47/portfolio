BridgeGen

Full-stack web application bridging Singapore's elderly and youth populations through collaborative features, gamification, and cultural exchange. Combats social isolation and the digital divide through meaningful intergenerational connections.

HTML JavaScript Bootstrap Python Flask

Core Features Developed

  • Dual User System: Separate youth and elderly interfaces with role-based authentication and NRIC validation
  • Tech Help Matching: Real-time request system where youth volunteers assist elderly users with technology issues
  • Cooking Game: Cooking Fever-style interactive minigame featuring 25 culturally significant recipes
  • Points & Rewards System: Dual currency economy (XP and Reward Points) with exchange mechanisms
  • Calendar & Events: Comprehensive scheduling system to coordinate intergenerational activities
  • Voice Commands: Hands-free navigation with 8 voice-activated shortcuts for accessibility
  • BridgeBot: Built-in AI assistant providing contextual help throughout the application
  • Email Verification: Secure OTP-based email authentication system
  • Admin Dashboard: Backend management system for user oversight and content moderation

Technical Stack

HTML5 CSS3 JavaScript Bootstrap Python Flask SQLite Google Maps API

Screenshots

Overall Value & Impact

BridgeGen combats social isolation and the digital divide by creating meaningful connections between Singapore's youth and elderly populations. Through gamification, rewards, and accessible design, it transforms volunteering into an engaging experience while giving elderly users confidence to navigate an increasingly digital world.
